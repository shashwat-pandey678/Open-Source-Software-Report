# Open-Source-Software-Report

Description

Open-source software is often discussed as if it were only a technical model, but in reality it is also a social idea. It reflects the belief that knowledge grows when it is shared, inspected, improved, and passed forward instead of being locked behind ownership barriers.This report studies Git as a real example of that idea because Git is not just a version control tool; it is one of the clearest cases where a practical engineering crisis led to a community-driven solution that changed software development everywhere.
This project follows the report structure in the assignment brief by covering Git’s origin and philosophy, its Linux footprint, its place in the wider FOSS ecosystem, and a comparison with a proprietary alternative. The writing is intentionally explanatory and reflective, because the assignment rewards original thinking, depth, and technical understanding rather than copied definitions.
